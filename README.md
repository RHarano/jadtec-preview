# JADTEC サイトリニューアル — 確認用プレビュー

株式会社日本建築製図技術センター（JADTEC）コーポレートサイト リニューアルの**確認用プレビュー**です。

- これは WordPress テーマから書き出した**静的スナップショット**です（デザイン・レイアウト・動きの確認用）。
- **お問い合わせフォームの送信は動作しません**（PHPが動かないため）。本番のWordPressでは、確認画面・自動返信メールまで動作します。
- 検索エンジンには登録されません（全ページ noindex ／ robots.txt で全面拒否）。
